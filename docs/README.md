## 🏗️ **Project Architecture**

```
FundiMatch/
├── 📱 Frontend (React + Vite)
│   ├── Authentication (Firebase)
│   ├── Role-based Dashboards
│   └── Real-time Notifications
├── 🖥️ Backend (Flask + SQLAlchemy)
│   ├── RESTful API
│   ├── Database Models
│   └── Google Auth Integration
├── 🗄️ Database (SQLite → PostgreSQL)
│   ├── Users & Roles
│   ├── Jobs & Bookings
│   └── Notifications
└── 🚀 Deployment (Render)
    ├── Web Services
    ├── Static Sites
    └── Managed Database
```
